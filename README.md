## Sobre 
Uma aplicação web chamada Mychine, permite alugar equipamentos como pequenos tratores, cavadeiras e 
outros materiais tipicamente utilizados na construção civil. Todo o processo de aluguel é 
feito de forma online facilitando o serviço para o locatários e o locadores.

### Pre-Requisitos

- [Ruby](https://www.ruby-lang.org/en/documentation/installation/)
- [Bundler](https://bundler.io/)

### Execução
1. Instale as dependencias.
```
bundle install
```

2. Inicie a aplicação.
```
bundle exec jekyll serve --livereload
```

A aplicação estará rodando em http://localhost:4000/.
